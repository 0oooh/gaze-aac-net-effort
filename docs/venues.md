# Venues (reference)

Target publication venues for this work. The field's center of gravity is **ACM
(HCI / accessibility / eye-tracking)**, not IEEE. Used for writeup framing and
for showing target labs we understand their publication stage.

## Bullseye (where the target labs publish)

- **ACM CHI** — HCI flagship, top tier. Wobbrock, Findlater, Diyi Yang.
- **ACM ASSETS** (SIGACCESS) — the accessibility / AAC venue. Most on-target.
- **ACM ETRA** — the eye-tracking venue. Gaze typing, dwell, gaze methods.

## Adjacent (LLM / systems / NLP angle)

- **ACM IUI** — intelligent UIs; LLM + interface (KWickChat was IUI).
- **ACM UIST** — novel interface systems.
- **ACL / EMNLP / NAACL** — the prediction-modeling (NLP) framing
  (SpeakFaster precursor = NAACL; char-based AAC = EMNLP Findings).
- **SLPAT** workshop — Speech and Language Processing for Assistive
  Technologies; niche but exactly on-topic.

## Journals

- **ACM TACCESS** (Transactions on Accessible Computing), **ACM TOCHI**.

## IEEE (not the home, listed for completeness)

- **IEEE TNSRE** (rehab/BCI, motor disability), **IEEE VR** (gaze in VR).
  Adjacent only; this project's home is ACM.

## Realistic tier for this project (undergrad, workshop/preprint goal)

- **CHI Late-Breaking Work (LBW)**, **ASSETS poster**, **ETRA short** —
  achievable solo tracks.
- **arXiv preprint** + **SLPAT workshop** — easiest and most on-target.
